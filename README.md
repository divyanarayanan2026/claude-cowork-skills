# claude-cowork-skills

Custom skills for [Claude Cowork](https://claude.ai) — installable bundles that extend Claude's capabilities for specific workflows.

## What are skills?

Skills are instruction bundles that teach Claude how to handle specific tasks — like analyzing data, creating documents, or managing workflows. Install them in Claude Cowork and Claude will automatically use them when relevant.

## Skills

### 📊 data-analyzer
Load any CSV or Excel file and get a polished dark-mode dashboard report with charts, KPI cards, key findings, statistics, and data quality info — all in a single self-contained HTML file.

**Triggers when you say things like:**
- "Analyze my sales data"
- "What's in this CSV?"
- "Make me some charts from this spreadsheet"
- "Show me trends in this data"

**Output:** A beautiful HTML report with gradient charts, KPI cards, color-coded findings, and full summary statistics.

**Install:** Download `data-analyzer.skill` and drag it into your Claude Cowork skills folder.
